What's new in this last version?

New features in 1.0:

  • Improved write support:
    • Now permits renaming drives, projects and songs
    • Locate points can now be set and saved from within HD24connect
    • Permits creating empty drive images in files
    • Permits formatting drives and drive image files
    • Allows song creation with alternative track counts
    • Allows write protecting/write enabling songs
    • Allows pre-recording silence into songs by specifying a song length on song creation; this can prevent the need for recovery when a recording suffers a power failure*
    • Automatic song lengthening as required, if space allows*
  • Improved recovery features:
    • Improved drive detection algorithm works around accidentally initializing drives in some cases
    • Certain classes of 'unexpected end of song' can now be repaired automatically
    • Improved do-it-yourself live crash recovery: now only requires you to specify the approximate song length of the lost recording
    • Integrated wavefix audio restauration for damaged audio on Invalid Superblock disaster scenario
  • User interface enhancements:
    • Recorder tab now displays song information
    • Improved catalog functionality now displays longer filenames and write protect status of songs
    • HD24connect now permits sorting the catalog, allowing you to change the order of the songs within a project
    • Mixer tab allows creating a fast mixdown of a song, saving and restoring mixer settings and saving mixdowns to file
    • Locate points are now simpler to enter (using keyboard rather than mouse)
* These functions should still be considered unsafe in the 1.0.0 alpha versions.
