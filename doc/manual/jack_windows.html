Using JACK under Windows

Jack support for Windows has recently become available. To use it with HD24connect, you will need to install jackdmp which can be found on http://www.grame.fr/~letz/jackdmp.html. Download a package that contains a binary version and unpack it (with WinZip or similar).

Inside the package, you will find folders bin and jack-[version]. Set up JACK from the bin\windows folder.

Once set up, you will need to make sure that HD24connect can access libjack.dll (possibly named libjackdmp.dll) by copying it to the HD24tools directory and naming it appropriately. To manage connections, you'll want to use the qjackctl.exe program.
